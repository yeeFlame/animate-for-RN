React Native 动画处理

一个RN的动画可以按照以下步骤进行

  1. 使用基本的Animated组件，如 Animated.View Animated.Image Animated.Text
  2. 使用Animated.Value设定一个或多个初始值
  3. 将初始化值绑定到动画目标的属性上
  4. 通过 Animated.timing等函数设定动画参数
  5. 调用start启动动画

以上的步骤是使用RN的api Animated 来实现的。但是实现动画可以通过其他途径达到

在RN实现动画的几种方式

  • setState
  • setNativeProps
  • LayoutAnimation
  • Animated
